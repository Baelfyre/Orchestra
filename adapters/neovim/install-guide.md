# Neovim Installation Guide

1. Use the scaffold metadata in `package.json` for packaging structure only.
2. Keep routing, governance, execution, manifest parsing, and audit behavior in the shared runtime core.
3. Apply [workspace-instructions.template.md](workspace-instructions.template.md) through Neovim local instruction files or plugin config where supported.
4. Treat this folder as packaging scaffolding, not a published registry package.
